Commbank Statement Converter (PDF to CSV)

This site takes PDF transaction statements from the Commonwealth Bank of Australia and converts them to usable formats like CSV and JSON

In addition, it offers post processing features like extracting out the purchase date on credit/debit card purchases and blending multiple statements together

I was only able to design it using my own bank statements, I assume Commbank have a consistent statement format but if not please raise an issue here

If you have questions, encounter bugs or want to request features, you can raise them here: https://github.com/alshdavid/Commbank-statement-converter/issues

Notes on Security

Given the sensitivity of the information in bank statements, this website takes the following precautions to ensure there is as little risk as possible to you

  • The website processes everything within the browser, so your data never leaves your device

  • The website is entirely self contained and loads no external dependencies

  • This website is fully open source allowing for security audits and contributions. View the code here https://github.com/alshdavid/Commbank-statement-converter
